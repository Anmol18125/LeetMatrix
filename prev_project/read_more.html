Differences between XML and HTML

XML HTML
eXtensible Markup Language HyperText Markup Language
Used to transport and store data Used to display data
Strict syntax rules More flexible syntax rules
Data is stored as plain text Content is displayed as web pages
Tags are case-sensitive Tags are not case-sensitive
Self-descriptive and extensible Predefined set of tags
Allows nesting of tags in a strict manner Allows nesting of tags in a flexible manner
Primarily used for data interchange Primarily used for data presentation
